# Mus

Spanish card game. 

## Usage

1. Install npm dependencies:
```
sudo npm install
```

1. Install bower dependencies:
```
bower install
```

1. Run:
```
gulp
```

1. Browse to:
```
http://localhost:3000/
```

## Development Hints

To run and watch the mus client alone:
```
gulp watch
```

and browse to:
```
http://localhost:8000/
```

To run the node server as well:
```
gulp nodemon
```

and browse to:
```
http://localhost:3000/
```

## Testing

To run tests on the NodeJS server application:
```
gulp test-server
```

To run unit tests on the client application:
```
gulp test-client
```
